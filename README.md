# Apollo Care Companion

## Project Description
Apollo Care Companion is a healthcare web application designed to help patients manage their healthcare needs efficiently. The application allows users to book doctor appointments, set medicine reminders, and maintain personal health records through a simple and user-friendly interface.

## Features
- User Registration and Login
- Doctor Appointment Booking
- Medicine Reminder
- Health Records Management
- Emergency Contact Support
- User-Friendly Interface

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Installation Guide
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

## How to Use
1. Open the application.
2. Register or log in.
3. Book doctor appointments.
4. Set medicine reminders.
5. View and manage health records.
